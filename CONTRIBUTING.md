# Contributing

欢迎贡献改进。

## 可以贡献什么

- 修正事实错误、术语翻译或排版问题
- 对照原视频补充遗漏内容
- 增加公式推导、图示、代码练习
- 添加每章 quiz、复习卡片或学习路线
- 改善中文表达，让教程更适合自学

## PR 建议

1. 每个 PR 尽量聚焦一个主题或一讲。
2. 如果修改课程事实，请尽量附上视频时间戳或课程链接。
3. 保留关键英文术语，例如 `tokenization（标记化）`、`KV cache`、`FSDP`。
4. 不要加入未经核实的课程结论；如果是背景补充，请标注“补充说明”。
5. 避免上传大体积视频、音频或完整第三方版权材料。

## 文风约定

- 面向自学者，先讲直觉，再讲公式。
- 尽量解释“为什么”，不只列概念。
- 对工程主题，说明瓶颈、权衡和排查方法。
- 对数学主题，说明变量含义和适用范围。
